### Приступая к работе

1. Склонируй и запусти этот проект: `git clone git@github.com:vladdof/me-story.git`;
2. `cd me-story` чтобы перейти в директорию с проектом после клонирования;
3. Чтобы установить зависимости проекта, выполни команду `yarn install` или `yarn`;
4. Запуск проекта `yarn serve`;
5. Если не открылся браузер то перейди в браузер и открой страничку [http://localhost:3000](http://localhost:3000).


&nbsp;
### `yanr build`

Команда выполнит сборку в папку `dist`.\
Связываются все зависимости и происходит оптимизация кода.

Сборка минифицируется, а имена файлов содержат хеши.\
Приложение готово для размещения.


&nbsp;
### Приступая к работе с сервером

1. `cd server` чтобы перейти в директорию с файлами для сервера;
2. Чтобы установить зависимости выполни команду `yarn install` или `yarn`;
3. Запуск сервера `node index.js`;
4. В консоли увидишь уведомление о запуске
